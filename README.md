Приложение для доставки продуктов "ФрешМарт".

1. Введение.
Приложение "ФрешМарт" создано для заказа и доставки свежих продуктов
прямо до двери пользователя.
Цель приложения - создать удобный, быстрый и понятный способ заказывать
продукты онлайн, не выходя из дома.

2. Цели и задачи.
Цели:
  1) Реализовать удобный каталог товаров с категориями и корзиной.
  2) Создать простой и привлекательный интерфейс на русском языке.
  3) Добавить выбор адреса доставки через интерактивную карту.
  4) Обеспечить быстродействие и стабильность приложения.

3. Функциональные требования.
Пользовательский интерфейс:
  1) Простой, интуитивный и минималистичный дизайн.
  2) Регистрация и вход в аккаунт по email и паролю.
  3) Каталог товаров с фильтрацией по категориям.
  4) Корзина с подсчётом суммы заказа.
  5) Выбор адреса доставки на карте (OpenStreetMap).
  6) Выбор способа оплаты: карта, наличные, СБП.
  7) Поддержка доставки по России и международной доставки.

4. План разработки.
   1) Проектирование интерфейса и дизайна.
   2) Разработка экранов приложения.
   3) Реализация карты и геокодирования адреса.
   4) Тестирование.

5. Запуск.
   1) Публикация приложения в Google Play.
   2) Маркетинговая кампания.

6. Поддержка.
   1) Обновление приложения.
   2) Техническая поддержка пользователей.

Платформы: Android.
Язык: Русский.

Заключение: Создание "ФрешМарт" покажет, насколько удобным может быть
приложение для заказа продуктов, с помощью которого можно в пару нажатий
выбрать товары, указать адрес на карте и оформить доставку.
